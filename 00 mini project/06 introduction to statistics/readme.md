### Keterangan dataset
     - customer_id: ID unik untuk setiap pelanggan.
     - age: Usia pelanggan (numerik).
     - gender: Jenis kelamin pelanggan (kategorikal: 'Pria', 'Wanita').
     - annual_income: Pendapatan tahunan dalam juta Rupiah (numerik).
     - time_on_site_minutes: Rata-rata waktu yang dihabiskan di situs web per bulan (numerik).
     - items_purchased_monthly: Rata-rata jumlah item yang dibeli per bulan (numerik, diskrit).
     - has_subscribed: Apakah pelanggan berlangganan newsletter (kategorikal/biner: 1 untuk ya, 0 untuk tidak).
